Explore the Top 3 Counties With Libraries in New York

By: Lina Battikha

This webpage was created to explore the different resources available the top three counties in New York that have most number of libraries. In fact, at the time that this data was collected, Suffolk County had the most number of libraries within any county in the United States. 📚

The data was used throughout this project was gathered from the U.S. Census Bureau is the data collection agent for IMLS Public Libraries Survey. This data is from 2014. Future goal would be to update this webpage to include more recent data collection regarding the counties' libraries and explore that changes that have occurred.

You can explore the different counties by changing the library selection right below! The default library selection is SUFFOLK county.

Distribution of Libraries Within Their Counties

The map below provides a visualization of the location of where the counties are on the map of New York as well as where each of the libraries are located. If you hover over any point, you can learn information the library that includes the library's name, employee hours and number of librarians.

The number of librarians is calculated based off of the number of librarians per 25,000 within the service area population. The number of Employee hours is the total number of hours of all the employees within a pay period (2 weeks).

Suffolk Leading in Number of Libraries

The Collections are Endless

Let's take a look at the collections each library has! The number of collections and the variablity within the types of the collections that are available can indicate the library's service population as a larger population leads to more funding, so more money can be splurged onto different collections 😁.

Explore the different libraries and the number of collections they have within each collection category by hovering over any bar on the graph!

Who's Popular?

Let's take a look at the distribution of the number of library visits and their registered users for each library within the selected county. Hover over any point to get some numbers! Regardless of the county you've chosen, as expected, there is a positive linear trend between number of visitors and registered users within the year 📈.

It's All Connected

Now let's take a closer look at resources that are shared between libraries within this county. The network diagram focuses on all the connections that are made between libraries that also have Print Collections. So, if you ever need to find another library that has print collections within this counties, you are set. 🕸

Take a closer look! You can zoom into this diagram and over a line to see the link between the two libraries. Hover over nodes of the diagram to retrieve the library name.
